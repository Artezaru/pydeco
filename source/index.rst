pydeco documentation
====================
Welcome to pydeco's documentation!

``pydeco`` is a Python package that provides tools to create and use decorators.
The API of the package is described in the documentation :doc:`./api` section.

The package ``pydeco`` also provides some decorators ready to use.
A User Guide is available in the documentation :doc:`./usage` section.
