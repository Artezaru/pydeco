pydeco
======

The package ``pydeco`` is composed of the following functions, classes and modules:

- ``pydeco.Decorator`` class is the base class for all decorators.
- ``pydeco.class_propagate`` function is used to propagate a decorator to all methods of a class.
- ``pydeco.decorators`` module provides some decorators ready to use.
